Conditional Colocalization Graphical User Interface

App Development - 2022

Colocalization analysis is a cornerstone technique in cell biology for the analysis of microscope images. It provides information on the localization of molecules within various subcellular compartments. The majority of colocalization analysis developments are only applicable to two-color microscopy images and are only able to investigate pairs of molecular entities. A member in the lab previously developed an approach for analyzing the colocalization relationships between three molecular entities called, "Conditional Colocalization Analysis". The approach included a series of functions that were written in MATLAB that analyze and visualize the results of three-color conditional colocalization, but these functions are processed using the command line interface, or console. A command line interface is not very user-friendly and may be difficult to use. Creating a GUI will allow the functions to be more interactive and easier to understand.
